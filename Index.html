 Payment Setup

          

            
🇧🇩 bKash/Nagad: Add your merchant number in Settings

            
🌍 International: Open Payoneer account → get US bank details → add in Settings

            
💳 Stripe: Create account at stripe.com → add API key

            
🅿️ PayPal: Add your PayPal merchant email

            
₿ Crypto: Add your USDT TRC20 wallet

          

        

        

          
 How to Go Live (Global)

          

            

              
Step 1 — Firebase Hosting

              

                1. Install Node.js

                2. npm install -g firebase-tools

                3. firebase login

                4. firebase init hosting

                5. firebase deploy

                → yoursite.web.app
              

            

            

              
Step 2 — Custom Domain

              

                1. Buy domain at GoDaddy

                2. Firebase Console → Hosting

                3. Add Custom Domain

                4. Copy DNS records

                5. Add to GoDaddy DNS

                → yourshop.com
              

            

            

              
Step 3 — Payment

              

                1. Open Payoneer account

                2. Get US bank details

                3. Add to Stripe/PayPal

                4. Add keys in Settings

                5. Test with $1 payment

                → Start earning!
